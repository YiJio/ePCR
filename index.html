Project Description

The electronic patient care reporting (ePCR) is a web and mobile application for the Rescate emergency response team of San Carlos Mexico. The purpose of the applications is to allow the rescue team to create, store, query, and perform analysis with patient records. The system provides a graphical user interface for the input of patient records. Patient records cannot be deleted but only notes can be added. The records are stored in a database with various information related to the patients. The system allows for the querying of patient records with the data that is stored in the database. With the data, summary report generation and statistical analysis can be performed. The summary reports can be created with two date ranges that summarize the patient data in that time frame. The statistical analysis portion of the system allows for the performace of trend call analysis to determine the busiest times. The system utilizes authentication and role-based access. All users must be added by an admin user before they can register and log in. The admin user(s) have more permission than the standard user(s). The admin user(s) can delete users, add users, create summary reports, and elevate the permissions of a user.
